# Instructions 

## Environment Setup in VSCode
### Create Virtual Environment
* `python -m venv venv`
* `venv\Scripts\activate`

### Install libraries
* When the virtual environment is activated:
    - `pip install pandas numpy seaborn`

### Setup Kernal
* In upper right hand corner click `Select Kernal`
* `Python Environments...`
* Then select the virtual environment that you created above
